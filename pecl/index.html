Registering the channel: 
pear channel-discover tarantool.github.io/tarantool-php/pecl
Listing available packages: 
pear remote-list -c Tarantool-PHP
Installing a package: 
pear install Tarantool-PHP/package_name
Installing a specific version/stability: 
pear install Tarantool-PHP/package_name-1.0.0
pear install Tarantool-PHP/package_name-beta
Receiving updates via a feed: 
http://tarantool.github.io/tarantool-php/pecl/feed.xml

Tarantool - PHP extension for Tarantool

PECL PHP driver for Tarantool Tarantool is an in-memory database and Lua application server. This package provides PECL PHP driver for Tarantool.

Install commandpecl install Tarantool-PHP/Tarantool
